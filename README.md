### Technologies
1. Swift 6
2. WebSocket
3. URLSession
4. Async/await
5. SwiftUI
6. AsyncStream
7. APNs
8. Core Data

### Goals to achieve
1. Learn webSocket
2. Deal with Swift 6
3. Centralise UI flow in one component
4. MVVM, separate UI logic from business logic
5. Build server side with Swift Vapor
6. Use decorator for refresh token logic and caching
